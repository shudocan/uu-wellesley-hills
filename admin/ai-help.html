Need a bigger change? 🚀

The ✨ Build a page tool in the editor is great for adding simple new pages. But some things are bigger — a redesign, a new feature (an online form, event registration, a members login, payments), restructuring the menus, or fixing something that looks broken.

For those, you connect a more powerful AI (or a person) to the website's code. Here's how — no coding required from you.

Option A — Claude Code Recommended

Claude Code can read this entire website, make changes, and show them to you before anything goes live.

  1. Go to claude.com/code (or open the Claude app) and sign in.
  2. Connect it to this website's code repository:
    https://github.com/shudocan/uu-wellesley-hills
  3. Describe what you want in plain English — for example: “Add an online volunteer sign-up form to the Engage page,” or “Redesign the homepage hero to feature this Sunday's service.”
  4. Review what it proposes. When you're happy, it can publish the change (or open it for approval).

Tip: paste the website summary below into your first message so the AI instantly understands how the site is built.

Option B — ChatGPT, Gemini, or another AI

  1. Open your AI of choice.
  2. Paste the website summary below, then describe the change.
  3. It will give you updated files or instructions — hand those to whoever manages the site (or Claude Code) to publish.

A few safety notes

  • Bigger changes should always be previewed/reviewed before going live.
  • Every change is saved in history — anything can be undone.
  • True features that need accounts, payments, or private data (logins, online payments) usually need a developer's help — an AI can draft them, but they should be set up carefully.

When in doubt

Reach out to Andrew or the office. It's always okay to ask before making a big change.
